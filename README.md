# Technno Lab 🌐

Landing page corporativa desarrollada con **HTML5, Bootstrap 5 y CSS personalizado**, como proyecto académico enfocado en la integración de componentes y diseño responsivo.

---

## 📌 Descripción

Technno Studio es una página web tipo landing diseñada para simular el sitio corporativo de una agencia tecnológica especializada en desarrollo web y soluciones digitales.

El proyecto demuestra la correcta implementación del framework Bootstrap, aplicando:

- Sistema de grid
- Componentes interactivos
- Diseño responsivo
- Buenas prácticas de maquetación y estructura semántica

---

## 🚀 Tecnologías utilizadas

- HTML5  
- Bootstrap 5.3  
- CSS3  
- Bootstrap Icons  

---

## 🧩 Componentes de Bootstrap implementados

Se utilizaron los siguientes componentes:

- Navbar (barra de navegación responsiva)  
- Cards (presentación de servicios y proyectos)  
- Modal (visualización de información adicional)  
- Form (sección de contacto)  
- Buttons (botones estilizados con clases de Bootstrap)  

---

## 📱 Responsividad

La página fue desarrollada utilizando el sistema de grid de Bootstrap para adaptarse correctamente a diferentes tamaños de pantalla:

- Dispositivos móviles (`col-12`)  
- Tablets (`col-md-6`)  
- Escritorio (`col-lg-4`)  

Se emplearon clases como:

- `container`  
- `row`  
- `col-*`  
- `d-flex`  
- `align-items-center`  
- `justify-content-center`  

---

## 🏗 Estructura de la página

1. **Inicio (Hero Section)**  
   Presentación principal con texto e imagen alineados mediante el sistema de grid.

2. **Servicios**  
   Uso de Cards para mostrar los servicios ofrecidos.

3. **Portafolio**  
   Presentación de proyectos con acceso a información adicional mediante Modal.

4. **Contacto**  
   Formulario estructurado con clases `form-control` y `form-label`.

5. **Footer**  
   Pie de página con enlaces e íconos de redes sociales.
